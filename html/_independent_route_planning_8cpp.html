 Calculates the shortest path and an alternative path between two vertices in a graph. The time complexity of the function is O((V + E) log V) for each call to Restricted Dijkstra, where V is the number of vertices and E is the number of edges in the graph. Since Restricted Dijkstra is called twice (once for the best path and once for the alternative path), the overall complexity is O(2 * (V + E) log V), which simplifies to O((V + E) log V).
